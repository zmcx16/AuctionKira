# AuctionKira (拍賣網瀏覽器擴充套件)

# 功能
  * 關鍵字過濾功能
  * 賣家過濾功能
  * 去除廣告
  * 預載頁面
  * 自動執行

# 支援網站
  * 買動漫(https://myacg.com.tw/index.php)
  * 露天拍賣(https://www.ruten.com.tw/)
  * 蝦皮拍賣 - TBC
  * YAHOO拍賣 - TBC
